# 📊 Customer Segmentation & Churn Pattern Analytics in European Banking (Excel Dashboard)

## 📌 Project Overview

Customer churn is one of the biggest challenges in the banking industry because losing existing customers directly impacts revenue and business growth. This project analyzes customer behavior using Microsoft Excel to identify the major factors influencing customer churn across different customer segments.

The dashboard provides an interactive view of customer demographics, financial characteristics, and engagement levels, helping stakeholders understand churn patterns and make data-driven retention strategies.

---

# 🎯 Project Objective

The primary objective of this project is to:

- Analyze customer churn using historical banking data.
- Segment customers based on demographic and financial characteristics.
- Identify high-risk customer groups.
- Measure churn across different customer segments.
- Build an interactive Excel dashboard for business decision-making.
- Provide actionable insights to improve customer retention.

---

# 📊 Business Problem

European banks are experiencing customer attrition, resulting in financial losses and declining customer loyalty.

The challenge is to determine:

- Which customers are leaving?
- Why are they leaving?
- Which customer segments have the highest churn?
- Which regions require immediate attention?
- Which high-value customers are at risk?

---

# 💡 Solution

Using Microsoft Excel, customer data was cleaned, transformed, segmented, and analyzed using Pivot Tables, Pivot Charts, KPIs, and Slicers.

The dashboard provides interactive visualizations that help decision-makers identify churn patterns quickly.

# 📌 Key Insights

Some important insights generated from the analysis include:

- Germany has the highest churn rate.
- Active customers have significantly lower churn.
- Customers with higher balances show higher revenue risk.
- Middle-aged customers contribute most to churn.
- Credit score influences customer retention.
- Customer engagement plays an important role in reducing churn.
